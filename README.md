# Zap Calendar

Integration between WhatsApp Web and Google Calendar built with Node.js. This integration fetches
today's events from Google Calendar and sends them to the desired contact via WhatsApp Web.

## Used technologies

 - Node.js;
 - Selenium.

## Running the intregration

Before scheduling the integration with a cron job, you need to run the command below manually to log in
to both WhatsApp Web and Google Calendar:

```bash
npm start [phone_number_here]
```
